VIM plugin for HTML and CSS hi-speed coding
===========================================

What
----
zencoding for vim

Install
-------
Copy zencoding.vim to your ftplugin directory

    mkdir ~/.vim/ftplugin/html
    cp zencoding.vim ~/.vim/ftplugin/html/.
